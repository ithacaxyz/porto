# React Native Mode

## 1. Setup

```sh
pnpx gitpick ithacaxyz/porto/tree/main/examples/react-native-mode porto-react-native-mode && cd porto-react-native-mode
```

## 2. Prerequisites

- XCode for iOS
- Android Studio for Android

## 3. Install Dependencies

```sh
pnpm i
```

## 4. Run the app

### iOS

```sh
pnpm expo run:ios
```

### Web

```sh
pnpm start
```
