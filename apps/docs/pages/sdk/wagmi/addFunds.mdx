# `Actions.addFunds`

VanillaJS action for adding funds to an account. Uses Viem's [`addFunds`](/sdk/viem/WalletActions/addFunds) under the hood.

## Usage

:::code-group

```ts twoslash [example.ts]
import { Actions } from 'porto/wagmi'
import { config } from './config'

const response = await Actions.addFunds(config, {
  value: BigInt('1000000000000000000') // 1 ETH
})
```

```ts twoslash [config.ts] filename="config.ts"
// [!include ~/snippets/wagmi/config.ts]
```

:::

## Parameters

### address

`Address | undefined`

Address of the account to add funds to. If not provided, the current account will be used.

```ts twoslash
import { Actions } from 'porto/wagmi'
import { config } from './config'

const response = await Actions.addFunds(config, {
  address: '0x...', // [!code focus]
  value: BigInt('1000000000000000000')
})
```

### chainId

`number | undefined`

Chain ID to add funds on. If not provided, the current chain will be used.

### connector

`Connector | CreateConnectorFn | undefined`

The connector to use. If not provided, the current connector will be used.

### token

`Address | undefined`

ERC20 token address to add funds in. If not provided, native token (ETH) will be used.

```ts twoslash
import { Actions } from 'porto/wagmi'
import { config } from './config'

const response = await Actions.addFunds(config, {
  token: '0x...', // USDC address // [!code focus]
  value: BigInt('1000000') // 1 USDC (6 decimals)
})
```

### value

`bigint | undefined`

Amount to add. The value should be in the smallest unit of the token (wei for ETH, or the token's smallest unit).

```ts twoslash
import { Actions } from 'porto/wagmi'
import { config } from './config'

const response = await Actions.addFunds(config, {
  value: BigInt('1000000000000000000') // 1 ETH in wei // [!code focus]
})
```

## Return Value

```ts
type AddFundsReturnType = {
  /** Transaction ID */
  id: `0x${string}`
}
```

### id

`Hex`

The transaction ID that can be used to track the status of the add funds operation.

## Example

### Adding ETH to an account

```ts twoslash
import { Actions } from 'porto/wagmi'
import { config } from './config'
import { parseEther } from 'viem'

const response = await Actions.addFunds(config, {
  value: parseEther('0.5') // 0.5 ETH
})

console.log('Transaction ID:', response.id)
```

### Adding ERC20 tokens

```ts twoslash
import { Actions } from 'porto/wagmi'
import { config } from './config'
import { parseUnits } from 'viem'

const USDC_ADDRESS = '0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48'

const response = await Actions.addFunds(config, {
  token: USDC_ADDRESS,
  value: parseUnits('100', 6) // 100 USDC (6 decimals)
})

console.log('Transaction ID:', response.id)
```
