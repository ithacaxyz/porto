# `usePermissions`

Hook for retrieving active permissions for an account.

Internally uses [`experimental_getPermissions`](/sdk/rpc/experimental_getPermissions).

<TryItOut
  exampleSlug="#example"
  fn={({ provider }) => provider.request({ method: 'experimental_getPermissions' })}
  transformResultCode={(code) => {
    return 'const response = ' + code
  }}
/>

## Example


```tsx twoslash
import { Hooks } from 'porto/wagmi'

function Example() {
  const result = Hooks.usePermissions()
// @log: {
// @log: ...
// @log: }
}
```

### Query Specific Account

You can query permissions for a specific account by passing an address:

```tsx twoslash
import { Hooks } from 'porto/wagmi'

function Example() {
  const result = Hooks.usePermissions({
    address: '0x...'
  })
// @log: {
// @log: ...
// @log: }
}
```
