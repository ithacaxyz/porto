# `experimental_revokeAdmin`

Revokes an admin.

## Request

```ts
type Request = {
  method: 'experimental_revokeAdmin',
  params: [{
    /** Address of the account to revoke an admin on. */
    address?: `0x${string}`
    /** ID of the admin to revoke. */
    id: `0x${string}`
  }]
}
```

## Example

```ts twoslash
import { Porto } from 'porto'

const { provider } = Porto.create()

await provider.request({ // [!code focus]
  method: 'experimental_revokeAdmin', // [!code focus]
  params: [{ id: '0x...' }], // [!code focus]
}) // [!code focus]
```
