# Router

Instantiates a server framework-agnostic router for Porto.

<center>
  <img src="/router.svg" alt="Router" />
</center>

## Usage

```ts
import { Router, Route } from 'porto/server'

const router = Router()
  .route('/auth', Route.auth(/* ... */))
  .route('/merchant', Route.merchant(/* ... */))
```
