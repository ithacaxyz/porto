# Chains

Porto extends the chain definitions from [viem](https://viem.sh/docs/chains/introduction) with additional contract address information specific to Porto functionality.

## Supported Chains

Porto currently supports the following chains with pre-configured contract addresses:

```ts twoslash
import { Chains } from 'porto'

// Available chains
Chains.anvil        // Local development chain
Chains.baseSepolia  // Base Sepolia testnet
Chains.portoDev     // Porto development testnet
```

## Chain Structure

Each chain in Porto extends the standard viem chain type with additional contract information:

```ts
type Chain = {
  // Standard viem Chain properties
  id: number
  name: string
  nativeCurrency: {
    name: string
    symbol: string
    decimals: number
  }
  rpcUrls: {
    default: {
      http: string[]
      webSocket?: string[]
    }
    // ...additional RPC URLs
  }
  blockExplorers?: {
    default: {
      name: string
      url: string
      apiUrl?: string
    }
    // ...additional block explorers
  }
  contracts: {
    // Porto-specific contract addresses
    delegation?: {
      address: `0x${string}`
    }
    // ...and any other contract addresses
  }
  testnet?: boolean
}
```

## Defining Custom Chains

You can define your own chains with Porto-specific contract addresses using the `define` function:

```ts
import { Chains } from 'porto'

const myCustomChain = Chains.define({
  id: 123456,
  name: 'My Custom Chain',
  nativeCurrency: {
    name: 'Ether',
    symbol: 'ETH',
    decimals: 18
  },
  rpcUrls: {
    default: {
      http: ['https://my-custom-chain.rpc.example.com']
    }
  },
  contracts: {
    // Porto delegation contract address on your chain
    delegation: {
      address: '0x123abc...'
    }
  },
  testnet: true // or false for mainnet
})
```

## Usage with Porto

When creating a Porto instance, you can specify which chain to use:

```ts
import { Porto, Chains } from 'porto'

// Create a Porto instance that uses Base Sepolia
const porto = Porto.create({
  chain: Chains.baseSepolia
})

// Or use the Porto Dev testnet
const portoDev = Porto.create({
  chain: Chains.portoDev
})
```

## Usage with Wagmi

The chains can also be used with wagmi when configuring your application:

```ts
import { createConfig, http } from 'wagmi'
import { porto } from 'porto/wagmi'
import { Chains } from 'porto'

export const config = createConfig({
  chains: [Chains.baseSepolia],
  connectors: [porto()],
  transports: {
    [Chains.baseSepolia.id]: http()
  }
})
```
