# `Storage.idb`

The [IndexedDB](https://developer.mozilla.org/en-US/docs/Web/API/IndexedDB_API) storage backend for Porto.

Porto uses [IndexedDB](https://developer.mozilla.org/en-US/docs/Web/API/IndexedDB_API) as the default storage in browser environments.

## Usage

```ts twoslash
import { Porto, Storage } from 'porto'

const porto = Porto.create({
  storage: Storage.idb()
})
```
