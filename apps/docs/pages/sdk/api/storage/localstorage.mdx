# `Storage.localStorage`

The [localStorage](https://developer.mozilla.org/en-US/docs/Web/API/Window/localStorage) storage backend for Porto.

## Usage

```ts twoslash
import { Porto, Storage } from 'porto'

const porto = Porto.create({
  storage: Storage.localStorage()
})
```

## Example

An example of combining `Storage.localStorage` with `Storage.idb`:

```ts twoslash
import { Porto, Storage } from 'porto'

const porto = Porto.create({
  storage: Storage.combine(Storage.localStorage(), Storage.idb())  // [!code focus]
})
```
