# `Storage.memory`

The memory storage for Porto, which stores data in-memory with no persistence.
Under the hood, `Storage.memory` uses a [`Map`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Map) to store data.

Porto uses `Storage.memory` as the default storage in non-browser environments.

## Usage

```ts twoslash
import { Porto, Storage } from 'porto'

const porto = Porto.create({
  storage: Storage.memory()
})
```

## When to use `Storage.memory`

- In browserless or serverless environments such as [Cloudflare Workers](https://developers.cloudflare.com/workers/) and [AWS Lambda](https://docs.aws.amazon.com/lambda/).
- For testing purposes.
